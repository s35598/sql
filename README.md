# 📊 SQL Learning Journey @ PJATK

Welcome to my SQL repository! This repository contains all the queries, database designs, and assignments I have completed during my studies at **Polsko-Japońska Akademia Technik Komputerowych (PJATK)**, as well as my personal practice projects.

## 🎯 Objective
The main goal of this repository is to track my progress in learning SQL, database management, and relational database concepts. 

## 📂 Repository Structure

Here is a breakdown of what you can find in this repository:

*   **`/01_Basic_Queries`** - Simple `SELECT`, `WHERE`, `ORDER BY` exercises.
*   **`/02_Joins_and_Aggregations`** - `INNER/OUTER JOINS`, `GROUP BY`, `HAVING` and aggregate functions.
*   **`/03_Subqueries`** - Single-row, multiple-row, and correlated subqueries.
*   **`/04_DDL_DML`** - Data Definition Language (`CREATE`, `ALTER`, `DROP`) and Data Manipulation Language (`INSERT`, `UPDATE`, `DELETE`) scripts.
*   **`/05_Projects`** - Larger school assignments including ER Diagrams and full database creation scripts.
